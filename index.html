1. Pengantar Extension Methods

Extension Methods adalah fitur Dart yang memungkinkan programmer menambahkan method baru ke class yang sudah ada tanpa harus mengubah kode asli class tersebut atau menggunakan pewarisan (inheritance).

2. Masalah Tanpa Extension Methods

Tanpa extension methods, programmer harus membuat fungsi biasa (utility functions) secara terpisah, sehingga pemanggilan kode menjadi kurang rapi, kurang intuitif, dan sulit dibaca.

Contoh tanpa Extension:

String capitalize(String text) {
  return text[0].toUpperCase() + text.substring(1);
}

3. Solusi dengan Extension Methods

Dengan extension methods, method dapat dipanggil langsung dari object sehingga kode menjadi lebih rapi dan intuitif layaknya fitur bawaan class.

Contoh Implementasi:

extension CapitalizeExtension on String {
  String capitalize() {
    return this[0].toUpperCase() + substring(1);
  }
}

4 & 5. Sintaks & Tipe yang Bisa Di-extend

Extension menggunakan keyword extension ... on .... Fitur ini bisa digunakan untuk tipe data bawaan Dart seperti String, int, List, Map, double, hingga custom class.

Contoh pada Integer:

extension NumberExtension on int {
  bool get isEvenNumber => this % 2 == 0;
}

6 & 7. Null Safety & Generic Extensions

Extension juga mendukung tipe nullable (seperti String?) dan generic extension yang memungkinkan extension digunakan fleksibel untuk semua tipe data.

Contoh Nullable Extension:

extension NullableString on String? {
  bool get isNullOrEmpty => this == null || this!.isEmpty;
}

8 & 9. Unnamed Extensions & Konflik Nama

Extension dapat dibuat tanpa nama (unnamed) jika hanya digunakan pada satu file secara langsung. Jika terjadi konflik nama method yang sama dari dua buah extension, kita bisa mengatasinya dengan memanggil nama extensionnya secara eksplisit.

Contoh Konflik Solusi:

MyExtension(text).printData();

10. Extension untuk Custom Classes

Extension juga dapat disematkan pada class buatan kita sendiri untuk memisahkan logika tambahan agar kode utama tetap bersih.

Contoh pada Custom Class User:

extension UserExtension on User {
  void sayHello() {
    print('Hello, my name is $name');
  }
}
