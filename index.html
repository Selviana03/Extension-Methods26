1. Pengantar Extension Methods

Extension Methods adalah fitur Dart yang memungkinkan programmer menambahkan method baru ke class yang sudah ada tanpa harus mengubah kode asli class tersebut atau menggunakan pewarisan (inheritance).

2. Masalah Tanpa Extension Methods

Tanpa extension methods, programmer harus membuat fungsi biasa (utility functions) secara terpisah, sehingga pemanggilan kode menjadi kurang rapi, kurang intuitif, dan sulit dibaca.

Contoh sebelum ada Extension:

String capitalize(String text) {
  return text[0].toUpperCase() + text.substring(1);
}
